# AppMetrica SDK Ad Revenue Auto Collection Adapter for AppLovin MAX v12 (ILRD)

**Optional module**.
Maven: `io.appmetrica.analytics:analytics-ad-revenue-applovin-v12-auto:${VERSION}`.

## Builds

### Assemble

`./gradlew :ad-revenue-applovin-v12-auto:assembleRelease`

### Publish to MavenLocal

`./gradlew :ad-revenue-applovin-v12-auto:publishReleasePublicationToMavenLocal`

### Tests

`./gradlew :ad-revenue-applovin-v12-auto:testReleaseUnitTest :ad-revenue-applovin-v12-auto:generateReleaseJacocoReport`

### Code style

`./gradlew :ad-revenue-applovin-v12-auto:lint :ad-revenue-applovin-v12-auto:ktlint :ad-revenue-applovin-v12-auto:checkstyleRelease`

### Check AAR API

`./gradlew :ad-revenue-applovin-v12-auto:aarCheck`

### Regenerate AAR API dump

`./gradlew :ad-revenue-applovin-v12-auto:aarDump`
