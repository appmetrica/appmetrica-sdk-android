# AppMetrica SDK Ad Revenue Auto Collection Adapter for IronSource v9

**Optional module**.
Maven: `io.appmetrica.analytics:analytics-ad-revenue-ironsource-v9:${VERSION}`.

## Builds

### Assemble

`./gradlew :ad-revenue-ironsource-v9:assembleRelease`

### Publish to MavenLocal

`./gradlew :ad-revenue-ironsource-v9:publishReleasePublicationToMavenLocal`

### Tests

`./gradlew :ad-revenue-ironsource-v9:testReleaseUnitTest :ad-revenue-ironsource-v9:generateReleaseJacocoReport`

### Code style

`./gradlew :ad-revenue-ironsource-v9:lint :ad-revenue-ironsource-v9:ktlint :ad-revenue-ironsource-v9:checkstyleRelease`

### Check AAR API

`./gradlew :ad-revenue-ironsource-v9:aarCheck`

### Regenerate AAR API dump

`./gradlew :ad-revenue-ironsource-v9:aarDump`
