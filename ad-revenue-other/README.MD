# AppMetrica SDK Ad Revenue Auto Collection Adapter for Other Networks

**Optional module**.
Maven: `io.appmetrica.analytics:analytics-ad-revenue-other:${VERSION}`.

## Builds

### Assemble

`./gradlew :ad-revenue-other:assembleRelease`

### Publish to MavenLocal

`./gradlew :ad-revenue-other:publishReleasePublicationToMavenLocal`

### Tests

`./gradlew :ad-revenue-other:testReleaseUnitTest :ad-revenue-other:generateReleaseJacocoReport`

### Code style

`./gradlew :ad-revenue-other:lint :ad-revenue-other:ktlint :ad-revenue-other:checkstyleRelease`

### Check AAR API

`./gradlew :ad-revenue-other:aarCheck`

### Regenerate AAR API dump

`./gradlew :ad-revenue-other:aarDump`
